- 通过jsonp跨域 

```js
//此处引用于组件中
export default{
    //-----  省略  --------
    
    //在这发起后端请求，拿回数据，配合路由钩子做一些事情，对于前端来说，钩子函数就是指再所有函数执行前，我先执行了的函数，即钩住我感兴趣的函数，只要它执行，我就先执行。 
    mounted(){   
        this.$nextTick( ()=>{
            var that = this;
            $.ajax({
                    type:"get",
                    dataType:"jsonp", //解决跨域
                    url:"接口地址",
                    data:{user_id:1},
                    success:function(data){
                        console.log(data);
                        that.newsList = data.result.list;
                    }
            })
        })
    }
    
}
```
